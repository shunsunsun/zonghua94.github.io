迭代器

  • 可以理解为容器（或string）的地址
  • 头：v.begin() 尾后：v.end()

  • 类型：const无法修改的是迭代器指向的对象的内容，而迭代器的地址时可以修改的

  • C++11新特性：cbegin() ; cend() 返回的是const类型的迭代器

  • 不能在对迭代器的范围for循环中改变vector中的元素（包括增删改），实际运行时，运行界面直接没了，但是没有任何报错？？

  • = < <=比较运算符只能对同一个容器的迭代器进行使用，比较前后关系。

  • ‘-‘符号运算返回的是difference_type（有符号整数），表示距离大小

数组

一维数组

  • 维度必须是一个常量表达式

  • 数组之间不能进行拷贝（vector容器可以）

  • 数组下标类型为无符号的size_t类型

  • 初始化操作中：

  • 可以取到超越数组维度的指针，但是谨慎使用（可以取尾后指针），取头和尾后指针现在有函数begin(arr)和end(arr)

  • 数组元素的指针可以执行所有迭代器运算符运算，两个不同数组的指针比较是无意义的

  • string vector等标准库类型，下表是无符号的，但是数组的下表可以为负数

  • 尽量不用字符数组表示字符串

  • 可以用数组初始化vector

  • 定义数组时，要制定数组的大小，也可以通过赋初值的方式隐式定义大小

初始化方式

  1. 拷贝初始化：用等号，或v1(v2)
  2. 直接初始化：用括号（括号内的东西是存储的内容）
  3. 列表初始化：用花括号
  4. 默认初始化：不赋初值
