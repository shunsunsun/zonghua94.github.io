位运算符

位运算符最好只对无符号类型进行操作，并且有 ‘|=’ ‘&=’ (类似+=)的使用方式

sizeof

  • sizeof返回size_t类型

  • sizeof只看类型不看变量合不合法，即使(*p)是一个无效的指针nullptr，也可以使用

  • sizeof不会把数组类型当做指针类型，对vector操作时，只会返回一个元素的大小，而不是整个容器的大小

显式转换

cast-name (expression)
castname:

  • static_cast:只要不包含底层const就可以用，比如int->double
  • const_cast:把底层const去掉
  • reinterpret_cast:可转换底层的指针类型，但是不要用！！
