熟悉C++（四）

发表于 2017-03-17 | 分类于 C++ |
C++的语句，包括switch、for、跳转、异常处理

switch语句

  • case关键字和它对应的值一起被称为case标签，case标签必须是整型常量表达式！

  • switch内部的变量定义：最好要加上花括号，且只在语句块内有效。

for循环语句

  • 如果想在for语句中定义多个变量，那么这些变量必须是有相同的基础类型。
  • 范围for循环：

expression必须是一个序列（比如：string vector 数组）的对象，这些对象的共同点是能返回迭代器的begin和end成员

  • vector容器对象中不能添加删除元素的原因：在范围for循环中预存了end()的值，一旦添加删除元素，end函数的值可能变为无效。

跳转语句

break

break语句的作用范围：最近的迭代循环语句 和 switch语句。
作用：跳出一层。

continue

continue语句的作用范围：最近的迭代循环语句。
作用：跳出这一次循环，进入下一次。

goto

尽量不用，程序可读性不好

异常处理

一旦抛出了异常，会终止当前的函数，寻找能处理发生这个异常的函数，如果没找到，则会转到名为terminate的标准库函数，这个函数与系统有关，怎么处理这个异常可能不一样。一般情况下会导致程序退出。
发生异常时，不会traceback，操作过的就操作过了，不会回退（像是没执行过这个函数一样）。
