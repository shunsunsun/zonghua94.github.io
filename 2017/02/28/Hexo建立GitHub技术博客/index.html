Hexo建立GitHub技术博客

发表于 2017-02-28 | 分类于 Hexo |
技术博客从这里开始

为什么要建立技术博客

  • 可以把平时的想法和积累以文字的形式保存下来，方便自己和其他人一起学习交流。
  • 而且还可以在找工作的时候给自己加分，嘿嘿

环境搭建

安装Git

下载桌面版GitHub，其实主要用到的是GitShell，不过桌面版的可以帮助使用GitHub还是不错的。
过程比较简单按照指示完成安装，不过其中遇到了一些问题：
不允许对应用程序文件盒程序集进行HTTP重定向。无法下载…
这个找了半天的原因，最后发现是因为校园网网络不好，挂了个代理重新下载就好了

安装Node.Js

下载安装Node.Js
记得在安装完成之后在环境变量Path中添加安装目录，使得’npm’命令生效

安装Hexo

打开’Git Shell’，使用’npm’命令进行安装

至此，本地博客搭建完成，接下来就要把博客部署到GitHub中了

在GitHub上部署博客

注册GitHub账号

在GitHub上输入账号，邮箱，密码完成注册

创建页面仓库

点击创建新的仓库

这里一定要注意命名格式！你的用户名和你建立的仓库名是一一对应的！

命名规则：github账号.github.io 一定要注意格式！
